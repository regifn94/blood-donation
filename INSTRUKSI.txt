► STEP 1: COPY CODE PYTHON
  Buka chat dengan Claude, scroll ke atas dan temukan artifacts:

  1. app/models.py          (~150 baris)
  2. app/database.py        (~60 baris)
  3. app/schemas.py         (~120 baris)
  4. app/auth.py            (~150 baris)
  5. app/main.py            (~450 baris)
  6. scripts/seed_database.py (~250 baris)

  Copy code dari setiap artifact ke file yang sesuai.
  PENTING: Hati-hati dengan indentasi Python!

═══════════════════════════════════════════════════════════

► STEP 2: INSTALL PYTHON & MYSQL
  - Python 3.9+ dari python.org
  - MySQL 8.0+ dari mysql.com

═══════════════════════════════════════════════════════════

► STEP 3: SETUP VIRTUAL ENVIRONMENT
  Buka CMD di folder backend, lalu jalankan:

  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt

═══════════════════════════════════════════════════════════

► STEP 4: BUAT DATABASE MYSQL
  Buka MySQL Command Line Client:

  CREATE DATABASE donor_darah_db;
  exit;

═══════════════════════════════════════════════════════════

► STEP 5: EDIT FILE .env
  Sesuaikan username, password, dan host MySQL Anda:

  DATABASE_URL=mysql+pymysql://root:password@localhost:3306/donor_darah_db
                                ^^  ^^^^
                                ubah ini sesuai MySQL Anda

═══════════════════════════════════════════════════════════

► STEP 6: SEED DATABASE
  python scripts\seed_database.py

═══════════════════════════════════════════════════════════

► STEP 7: JALANKAN SERVER
  python run.py

  Buka browser: http://localhost:8000/docs
